# 纸上江山 - 下载中心

> 局域网联机战棋游戏 | 纸上谈兵，指尖江山

纸上江山是一款支持局域网联机的战棋游戏，支持 **Android 手机/平板** 和 **浏览器网页版** 两种游玩方式。

## 下载方式

| 平台 | 下载方式 |
|------|---------|
| 📱 Android | 下载 `apk/` 目录下的 APK 安装包 |
| 🌐 网页版 | 直接打开 `game.html` 即可游玩 |

更多信息请访问 **[下载中心](https://你的用户名.github.io/仓库名)**。

## 联机说明

1. 所有设备连接 **同一个 WiFi / 局域网**
2. 在游戏中选择"创建房间"或"加入房间"
3. 开始对战！

## 技术栈

- 原生 HTML/CSS/JavaScript (Canvas)
- Capacitor (Android 打包)
- 局域网 WebRTC / Socket 通信
